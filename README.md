# foodify

A new Flutter Food Ordering/Delivery App (Still-in-progress) project.

## Features

* SplashScreen
* Home Page
* Show products based on category
* Shopping cart
* Place your order
* Track order
* User profile
* Sign in & Sign up
* User address location from google map
* Zone based order
* User authentication
* Firebase notification
* Payment(Paypal)
* Create new products on the backend
* See order details
* Update order status
